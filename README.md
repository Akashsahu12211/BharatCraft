# BharatCart - Premium Static Website

BharatCart is a premium multi-page static website that showcases Indian artisans, handmade products, and cultural craftsmanship.

## Tech Stack

- HTML
- CSS
- Vanilla JavaScript

No frameworks, no backend, and no Firebase/API dependency are used in the current implementation.

## Pages

- index.html
- products.html
- artisans.html
- product-details.html
- login.html
- signup.html

## Shared Assets

- css/style.css
- css/components.css
- js/ui.js
- js/main.js
- assets/images
- assets/icons

## Frontend Features

- Sticky responsive navigation with mobile menu
- Premium hero and card-based UI with glassmorphism accents
- Product search and category filtering
- Product details with static reviews
- Cart drawer + inline cart section on products page
- localStorage cart add/remove/increase/decrease
- Frontend form validation for login/signup
- Smooth scrolling, reveal effects, and staggered card animations

## Run Locally

Open index.html directly in a browser.

Optional: use any static server (for example, VS Code Live Server) for a smoother local preview.
